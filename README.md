# João Raphael Cunha - AI Engineer & Sound Designer Portfolio

Welcome to my professional portfolio. I am **João Raphael Cunha**, an AI Engineer and Sound Designer focused on building intelligent automation, robust data ecosystems, and immersive audio experiences.

## 🚀 About the Project
This portfolio is a bespoke, brutalist-inspired web experience designed to showcase the intersection of artificial intelligence and sound design. It features:
- **Interactive Project Hub**: Highlights my work in AI Automation, RAG Systems, and Enterprise Orchestration.
- **Embedded Audio Experience**: A custom-built player featuring my latest beats and sound design projects (YAHARD / Y42_BEATS).
- **Modern Tech Stack**: Built with a focus on performance and minimalist design using HTML5, Tailwind CSS, and Alpine.js.

## 🛠️ Tech Stack
- **Frontend**: HTML5, Tailwind CSS (Styling), Alpine.js (Interactivity).
- **Audio**: Custom integrated HTML5 audio engine.
- **Design**: Brutalist aesthetics with Glassmorphism and responsive layouts.
